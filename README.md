# 🌳 Родологическое дерево (Rodolog)

Веб-приложение для построения и просмотра семейного дерева. Позволяет добавлять родственников, устанавливать связи между ними и визуализировать дерево в удобном интерактивном виде.

---

## Возможности

### Управление людьми
- Добавление, редактирование и удаление людей
- Поля: фамилия, имя, отчество, пол, дата и место рождения, дата и место смерти, профессия, заметки
- Загрузка фото — отображается в карточке дерева и в попапе с деталями
- Если фото не загружено — показывается аватар в зависимости от пола

### Контактные данные
Для каждого человека можно указать:
- 📞 Телефон
- ✉️ Email
- 💬 WhatsApp
- ✈️ Telegram
- 🔵 ВКонтакте
- 📷 Instagram
- 👤 Facebook
- 🟠 Одноклассники

В режиме просмотра контакты становятся кликабельными ссылками с иконками.

### Связи между людьми
Поддерживаются три типа связей:
- **Родитель → Ребёнок** (синяя линия)
- **Супруги** (розовая пунктирная линия)
- **Братья / Сёстры** (зелёная пунктирная линия с U-образным маршрутом)

### Визуализация дерева
- Автоматическая расстановка по поколениям (временная шкала): более старшие — выше
- Супруги отображаются на одном уровне, дети — ниже родителей
- В карточке показывается текущий возраст (или возраст прожитых лет для умерших)
- Округлённые линии связи без пересечения карточек

### Интерактивность
- **Перемещение карточек** мышью — позиции сохраняются в `localStorage`
- **Масштабирование** колесом мыши или кнопками `+` / `−`
- **Панорамирование** перетаскиванием фона
- Поиск по списку людей

### Два режима полноэкранного просмотра
| | Режим редактирования | Режим просмотра |
|---|---|---|
| Кнопка | ⛶ Maximize | 👁 Eye |
| Перемещение карточек | ✅ | ❌ |
| Редактирование / удаление | ✅ | ❌ |
| Контакты кликабельны | ✅ | ✅ |
| Бейдж «Режим просмотра» | ❌ | ✅ |

```bash
npm install

```

Приложение откроется по адресу: [http://localhost:5173](http://localhost:5173)

## Хранение данных

Все данные хранятся в браузере (`localStorage`):

| Ключ | Содержимое |
|---|---|
| `rodolog_data` | Массивы людей и связей |
| `rodolog_tree_positions` | Пользовательские позиции карточек |

Фотографии хранятся в формате base64 внутри объекта человека.

## Скриншоты

### Дерево с аватарами
Карточки отображают аватар (фото или силуэт по полу), имя, годы жизни и возраст.

### Попап с деталями
При клике на карточку открывается окно с полной информацией, контактами и связями.

### Полноэкранный режим просмотра
Режим «только чтение» с бейджем «Режим просмотра» — удобен для демонстрации дерева.
